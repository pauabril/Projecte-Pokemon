# Projecte-Pokemon

## Controls

| Action | Key | Alt Key |
| --- | --- | --- |
| Move up | <kbd>W</kbd> | <kbd>↑</kbd> |
| Move down | <kbd>S</kbd> | <kbd>↓</kbd> |
| Move left | <kbd>A</kbd> | <kbd>←</kbd> |
| Move right | <kbd>D</kbd> | <kbd>→</kbd> |
| Interact | <kbd>E</kbd> | <kbd>Enter</kbd> |
| Open menu | <kbd>Q</kbd> | <kbd>Tab</kbd> |
| Quit game | <kbd>Esc</kbd> |  |
